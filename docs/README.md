# Documentation

This directory captures the standards and design notes for the AI Platform Foundation repository.

## Contents

- [Architecture](architecture.md): high-level system context and repository boundaries.
- [Development](development.md): local development workflow, validation commands, and contributor tooling.
- [Setup](setup.md): reserved for future environment-specific setup notes.
- [Architecture Decision Records](adr/): decisions that shape the portfolio foundation.

## Documentation principles

Keep documentation short, practical, and close to the automation it describes. When repository behavior changes, update the relevant document in the same pull request.
